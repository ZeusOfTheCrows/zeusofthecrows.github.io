remember to stay safe and have fun when browsing the web! to that end, here are some extensions that you may not have heard of, somewhat rated by usefulness

name website short description
shinigami eyes link see at a glance how transphobic/trans-friendly social network pages and users, by the colour of the link
sponsorblock link skip community sourced sponsor segments of youtube videos
search by image link reverse image search with multiple providers on right click. web archives by the same dev is also good
imagus link enlarge images / image links on mouse hover. particularly good for open [unstyled] directories
translate web pages link translate pages on the fly (albeit through yandex/google - project bergamot is getting there but not yet)
adaptive tab bar color link not hugely useful, but pretty: adapts browser chrome to web page theme colour
